**Задание 1. Написать многопоточный и асинхронный серверы, которые обрабатывают внешние запросы  пор страницам сайта https://dental-first.ru/catalog. Для тестирования напишите сервер, который отправляет множественные запросы по различным страницам сайта. Сами  сервера должны сохранять названия товаров в файл. Сравните время выполнения, потребляемую память, и нагрузку на сервер.
Дополнительно: добавить подсчет суммарной стоимости всех пропарсенных товаров. Найдите оптимальный вариант решения этой задачи для асинхронного кода.**

Решение находится в папке task1. Имеется три py файла:<br>
Асинхронный сервер - AsyncioServer.py,<br>
Многопоточный сервер - ThreadingServer.py,<br>
Клиент - client.py.<br>

Для начала работы необходимо запустить асинхронный и многопоточный серверы. Далее запускается клиент, который делает запросы на серверы. <br>
Сервер находит названия всех товаров передаваемой категории и записывает их в отдельный текстовый файл. Вычисляется затраченное время и потребление памяти для каждого сервера. Далее суммируется стоимость всех пропарсенных товаров. <br>
Асинхронный сервер занимает порт 8081<br>
Многопоточный сервер занимает порт 8082<br>

**Задание 2. Напишите асинхронный и многопоточный сокеты, и клиент для них, которые считывают количество строк в файлах некой директории. Сравните показатели потребления, и время выполнения. Определите, какая минимальная память гарантировано обеспечит обработку 1000 одновременных запросов**

Решение находится в папке task2. Имеется три py файла: <br>
Асинхронный сервер - AsyncioServer2.py, <br>
Многопоточный сервер - ThreadingServer2.py, <br>
Клиент - client2.py. <br>

Асинхронный сервер занимает порт 8081 <br>
Многопоточный сервер занимает порт 8082 <br>

Для начала работы необходимо запустить асинхронный и многопоточный сокеты. Далее запускается клиент, который передает серверам файлы на обработку. <br>
Клиент измеряет: <br>
Количество строк в каждом переданном файле <br>
Время выполнения запросов <br>
Потребление памяти <br>
Производительность при 1000 одновременных запросов <br>
Минимальную гарантированную память для обработки 1000 запросов <br>
